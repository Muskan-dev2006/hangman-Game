# hangman-Game
A command-line Hangman Game developed in Python using Object-Oriented Programming (OOP) principles. This project demonstrates how OOP concepts such as classes, objects, encapsulation, and methods can be used to structure a simple interactive game.
📌 Description

Hangman is a classic word guessing game where the player must guess a hidden word one letter at a time. Each incorrect guess reduces the number of remaining attempts. The goal is to guess the complete word before running out of attempts.

This project focuses on building the game using clean OOP design, making the code more modular, reusable, and easier to maintain.

⚙️ Features

Built using Python OOP concepts

Random word selection from a list

Interactive command-line gameplay

Tracks correct and incorrect guesses

Prevents repeated letter guesses

Displays the current progress of the word

Win/Loss condition handling

🧩 OOP Concepts Used

Classes and Objects – to represent the game logic

Encapsulation – game data and methods are organized inside classes

Methods – used to handle game actions such as guessing letters and checking results

Modular structure – separates game logic from execution

🎯 How the Game Works

The program selects a random word from a predefined list.

The player guesses letters one by one.

Correct guesses reveal the letter positions in the word.

Incorrect guesses reduce the remaining attempts.

The game ends when:

The player guesses the word successfully ✅

The player runs out of attempts ❌
